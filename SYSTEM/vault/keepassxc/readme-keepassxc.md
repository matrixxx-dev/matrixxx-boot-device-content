## KeePassXC-Anwendung
### Erstellen einer Datenbank
#. Öffnen Sie Ihre KeePassXC-Anwendung. Klicken Sie auf die Schaltfläche
   „Neue Datenbank erstellen“
#. Der Assistent zur Datenbankerstellung wird angezeigt. Geben Sie den
   gewünschten Datenbanknamen und eine kurze Beschreibung ein
#. Klicken Sie auf „Weiter“.
   Der Bildschirm „Verschlüsselungseinstellungen“
   wird angezeigt (keine Änderungen vorzunehmen)
#. Klicken Sie auf die Schaltfläche „Weiter“.
   Der Bildschirm „Datenbankanmeldeinformationen“ wird angezeigt
#. Klicken Sie auf Fertig.

### Öffnen einer vorhandenen Datenbank
#. Doppelklick auf die Datenbank
#. Geben Sie das Passwort für Ihre Datenbank ein.
#. Klicken auf OK. Die Datenbank öffnet sich.

### Schnelles Entsperren
- Nur unter Windows und macOS durch biometrische Authentifizierung

### Hinzufügen von TOTP zu einem Eintrag
- Durch die Speicherung von TOTP-Codes in derselben Datenbank wie das
  Passwort entfallen die Vorteile der Zwei-Faktor-Authentifizierung.
  Wenn Sie maximale Sicherheit wünschen, empfehlen wir, **TOTP-Codes in einer
  separaten Datenbank aufzubewahren**, die Sie nur bei Bedarf entsperren.
